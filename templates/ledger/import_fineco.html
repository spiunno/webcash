How it works
  • The "Movimenti" (checking) section is posted to the bank account, and the "Carte di credito" section is posted to the credit card account.
  • The counterpart account for each transaction is guessed from a past transaction with the same description on that account.
  • Unmatched transactions are posted to an "Imbalance-<currency>" account for you to recategorize.
  • Re-importing the same statement is safe — matching rows are skipped, not duplicated.
